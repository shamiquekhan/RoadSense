What is RoadSense?

A data-driven platform that evaluates 14,793 road segments across Thailand and Maharashtra, combining GPS probe data, road network geometry, and street-level imagery into a single Speed Safety Score.

⚠

Speed Alignment

Measures how far posted limits and actual traffic speeds exceed Safe System survivable thresholds per road context.

🚶

VRU Exposure

Estimates vulnerable road user risk using pedestrian fatality curves, land use, road class, and proximity to schools and markets.

🌄

Road Environment

Evaluates protective infrastructure coverage using Mapillary detections and CLIP zero-shot imagery classification as fallback.

Two Independent Scoring Engines

RoadSense runs two methodologies in parallel, cross-validating every result.

4-Component Composite

Limit Misalignment 30%
Operating Speed Risk 30%
VRU Exposure 25%
Traffic Volume 15%

Score = 0.30L + 0.30O + 0.25V + 0.15T. Safe System limits from WHO 2021 / ITF 2018 per road class and land use.

3-Module RoadSense

Module A — Speed Alignment 35%
Module B — VRU Exposure 35%
Module C — Road Environment 30%

SSS = 0.35A + 0.35B + 0.30C. Each module independently scores a different safety dimension.

Risk Tier Classification

Tier Score Range Action Required
Critical — Immediate Review ≥ 0.65 Immediate Safe System intervention
High — Priority Review 0.45 – 0.65 Schedule priority review
Moderate — Scheduled Review 0.25 – 0.45 Monitor and plan upgrades
Low — Monitor < 0.25 Maintain current approach

Built for Replication

RoadSense works with OpenStreetMap data alone. No proprietary data required.

Any Country

OSM road network + Overpass API for POIs = global coverage. The CLIP imagery fallback works even in countries with no street-level photo databases.

Minimal Data

A minimum viable deployment needs only OSM roads, speed limit tags, and 50+ spot speed measurements. No crashes data required — this is a preventive model.

Open Source

MIT-licensed. Config-driven scoring weights, Makefile automation, CI via GitHub Actions. Run it quarterly to track changes over time.
